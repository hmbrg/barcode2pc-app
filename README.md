## Usage
[Expo](https://expo.io/) must be installed on your machine. Then just use `npm install` to install all dependencies.
Use
```bash
npm start
```
to run the app on your preffered device or simulator.

## Architecture
### Libraries and Frameworks used

- Redux (Rematch)
